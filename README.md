**Whity-Darky**

Follow and collect the coins to get ur lost treasure

---

**About the Game**

**Whity-Darky** is a 2D platformer built with Godot Engine where every scattered coin leads you to be closer ti ur treasure.

As you progress, every level becomes more challenging with tougher platforming sections and enemies.

---
**how to play**

- Follow the coins.
- Jump across platforms and gaps.
- Avoid  the enemies
- open doors to go to the next level
- get ur lost treasure

---

**Levels**
--
The game consists of four levels of increasing difficulty.
There is also a fifth level which is the paradise path to ur lost treasure.
